### NvChad custom config

## 目录
```
./custom
├── plugins                                // 客制化插件
│   ├── init.lua                           // packer安装对应插件
│   ├── overrides.lua											 // 覆盖nvchad对应插件配置
│   └── ......
├── init.lua
├── chadrc.lua                           // 整合客制化配置项
└── mappings.lua                         // 客制化插件对应键位
```
